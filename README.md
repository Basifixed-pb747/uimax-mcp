## 🚀 What uimax-mcp does

uimax-mcp turns Claude Code into a UI review tool. It helps check web pages, find layout issues, flag accessibility problems, and suggest fixes. It can review a site the way a frontend developer would, then help you make the page better.

Use it when you want help with:
- UI checks
- accessibility checks
- front-end review
- page quality checks
- site behavior review

It is built for people who want clear feedback on a website or app UI without having to inspect everything by hand.
